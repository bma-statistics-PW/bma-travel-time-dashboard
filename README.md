# BMA Travel Time Dashboard (2568)

Dashboard วิเคราะห์เวลาในการเดินทางในพื้นที่กรุงเทพมหานคร  
จัดทำโดย กลุ่มงานสถิติและวิจัย กองนโยบายและแผนงาน 
สำนักการจราจรและขนส่ง กรุงเทพมหานคร

---

## 📊 ภาพรวมโครงการ
- วิเคราะห์ 52 สายทาง
- ครอบคลุมช่วงเร่งด่วน เช้า–เย็น
- เปรียบเทียบข้อมูลย้อนหลัง 9 ปี (พ.ศ. 2560–2568)

---

## 🔍 Key Insights
- ความเร็วเฉลี่ย: 20.7 กม./ชม.
- ขาออกเร็วกว่า ขาเข้า ~24%
- พื้นที่ชั้นในมีความหนาแน่นสูงสุด
